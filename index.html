Web Development

Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management.

Frontend Development

The part of a website where the user interacts directly is termed as front end. It is also referred to as the ‘client side’ of the application.

Backend Development

It is used to store and arrange data.Backend is the server side of a website.
